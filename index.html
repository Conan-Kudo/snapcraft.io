Snapcraft

The secure and easy way to build and package your applications for Linux.

Snaps work everywhere

Snaps are the best way to install an app on Arch, Ubuntu, Debian or your favorite Linux. They are secure and always up to date. Snaps are faster, easier, safer, and the update system is transactional so your app is never broken.

“hello-world” tour

This tutorial assumes Ubuntu 16.04 LTS on your desktop, cloud or server. You can also install snapd on other flavours of Linux, and when you do the experience is exactly the same.

Snap discovery and the global snap store

From the command line you can see the current featured snaps:

$ snap browse
telegram        1.2  Secure messaging system
…

Those are the featured snaps today across all devices. You can also
try ‘snap browse categories’ to see specific types of app.

You can see featured snaps in a particular category too:

$ snap browse categories
browser                 office
database                management
Containers              appservers
content-management      java

Try ‘snap browse ’

$ snap browse database
cassandra      3.2  Large-scale document store
...

If you know what you want, you can search the snap name or description:

$ snap find hello-world
Name              Version    Summary
audovia           3.2.4      Make music using JFugue MusicStrings
hello-world       6.0        Hello world example
ubuntu-clock-app  3.6+snap3  Ubuntu Clock for the Unity 7 desktop

Install the snap you want with a simple command

snap install hello-world

Release channels — stable, candidate, beta and edge

You can choose if you want to get development versions of an app, by selecting the preferred “channel”. By default, snaps are installed from the stable channel, but there is also a candidate channel which is “soon to be the stable release”, a beta channel and an edge channel.

Choose how much risk you want to take on this app and set the channel accordingly with --candidate, --beta or --edge when you install the snap.

Switch from stable to the beta channel to get the latest development release:

snap install hello-world --beta

Know where your software comes from — snap publisher and details

The info command shows which channels are available:

snap info hello-world
XXX

Always fresh — update fast and reliably

Get the latest version of all your snaps by running:

snap refresh
